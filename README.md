Event Management Website
A simple event management system built with HTML, CSS, and JavaScript, featuring modules for Users, Vendors, and Admins.

Features
User: Sign up, log in, browse events, and register for events.
Vendor: Register, create and manage events.
Admin: Approve/reject events, manage users and vendors.
Responsive: Works on both desktop and mobile.
